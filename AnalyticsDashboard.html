Project Overview

A full-stack analytics dashboard built with Next.js 15 and React 19, designed to help teams monitor user behaviour and conversion metrics across their product. The application processes and visualizes approximately 56,000 events to surface actionable insights.

Key features include a funnel analysis view to track drop-off at each conversion step, A/B test variant comparisons, time-based trend charts powered by Recharts, and a raw events viewer with filtering and pagination. Deployed to Heroku with optional scheduled data refresh.
