A targeted portfolio built for the Junior Applied AI Web Developer role at Trend Micro, Ottawa. Features a full consumer security landing page and an interactive UI component library — both styled to match Trend Micro's brand.

Project Overview

This portfolio was built specifically to demonstrate fit for Trend Micro's Ottawa team. The project consists of two deliverables: a consumer security landing page ("Sentinel") that mirrors the look and feel of Trend Micro's own marketing site, and an interactive component library showcasing DaisyUI components styled with a custom Trend Micro dark theme.

The landing page includes a live threat intelligence dashboard, animated scrollytelling sections, a fully interactive pricing tier picker (with Device Protection, Anti-Scam, and Identity Protection tabs), and a mock testimonials section — all matching Trend Micro's brand colours, typography, and product copy. The component library documents badges, alerts, forms, tables, and modals in a live browser environment. Both are deployed to Vercel.
